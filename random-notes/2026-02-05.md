1. **Designing Great Android Apps for Tablets**
	- Use a Flexible Layout: ConstraintLayout, Linear Layout
	- Provide multiple layouts for different screens
	- Understanding screen density:
		- pixel: fixed
		- dpi = dot per inch
		- dp = density independent pixels -> system auto scales the dp units based on the actual pixel density
		- base: 160 dpi = medium density -> 1dp = 1 pixel
	- 320 dpi screen: 1 dp = 2 pixel
	- Using qualifiers:
		- Smallest screen width qualifier: sw800dp
		- Tablet: 7 inch -> sw600dp
		- Tablet: 10 inch -> sw800dp
