**ðŸ”¹ Handling Exceptions**

Use try / catch / finally:

```

    riskyOperation()
} catch (e: Exception) {
    -1
} finally {
    println("Always executed")
}
```

- try can return a value.
    
- finally always runs, even if an exception occurs.
    

For resource cleanup, prefer:

```

```
