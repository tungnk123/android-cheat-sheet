1. Interface functions are **abstract by default**
Functions with a body are **concrete**
Properties in interfaces have **no backing field**
`abstract` keyword is unnecessary in interfaces
Use interfaces for contracts, abstract classes for shared state

2. **Content Provider Details**
- **What is Content Provider?**
- A **Content Provider** manages access to structured data (databases, files) and provides a standard interface for other apps to query/modify data.

- **Content URIs Used**

- **CallLog Content Provider:**

```
CallLog.Calls.CONTENT_URI
// URI: content://call_log/calls
```

- **Contacts Content Provider:**

```
ContactsContract.CommonDataKinds.Phone.CONTENT_URI
// URI: content://com.android.contacts/data/phones
```

- **Query Process**

- **1. contentResolver.query()** - queries the Content Provider

```
context.contentResolver.query(
    uri,          // Content URI to query
    projection,   // Columns to return
    selection,    // WHERE clause (null = all rows)
    selectionArgs,// WHERE clause args
    sortOrder     // ORDER BY clause
)
```

- **2. Projection** - columns to retrieve:

```
// CallLog columns
CallLog.Calls._ID          // Unique call ID
CallLog.Calls.CACHED_NAME  // Contact name (if exists)
CallLog.Calls.NUMBER       // Phone number
CallLog.Calls.TYPE         // Call type (incoming/outgoing/missed)
CallLog.Calls.DATE         // Timestamp in milliseconds
CallLog.Calls.DURATION     // Call duration in seconds

// Contacts columns
ContactsContract.CommonDataKinds.Phone.CONTACT_ID    // Contact ID
ContactsContract.CommonDataKinds.Phone.DISPLAY_NAME  // Contact name
ContactsContract.CommonDataKinds.Phone.NUMBER        // Phone number
ContactsContract.CommonDataKinds.Phone.PHOTO_URI     // Avatar URI
```

- **3. Cursor** - result iterator:

```
cursor?.use { 
    it.moveToNext()           // Move to next row
    it.getLong(columnIndex)   // Get Long value
    it.getString(columnIndex) // Get String value
}
```
