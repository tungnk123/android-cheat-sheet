equals() vs == vs ===
	Java: == vs equals(), === Not available
	Kotlin:
		== -> Value/content (calls equals)
		equals() -> Value/content
		=== -> reference quality

=> 	Java: use .equals() for value, == for reference.
	Kotlin: use == for value, === for reference.
